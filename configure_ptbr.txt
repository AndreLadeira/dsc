1 - Criar pasta para o projeto, abrir terminal, ir para a pasta
2 - Na pasta, executar: git clone https://github.com/AndreLadeira/dsc
3 - Abrir cada projeto. (File->Open File or Project) Em cada projeto

	- Ir na aba Projects (chave inglesa, à esquerda na IDE) -> Build
	- Trocar o build directory para <raiz>/ATSP/_builds/<pasta dada pelo QT>
	
		exemplo no masi: (pasta raiz /home/masi/src)
		
		projeto base:
		
		trocar 

			/home/masi/src/dsc/build-base-Desktop_Qt_5_12_1_GCC_64bit-Debug 
		para
			/home/masi/src/dsc/ATSP/_builds/build-base-Desktop_Qt_5_12_1_GCC_64bit-Debug

		projeto atsp_base:
		
		trocar:

			/home/masi/src/dsc/ATSP/build-atsp_base-Desktop_Qt_5_12_1_GCC_64bit-Debug
		para:
			/home/masi/src/dsc/ATSP/_builds/build-atsp_base-Desktop_Qt_5_12_1_GCC_64bit-Debug
			
			

	
	OBS: se for compilar em modo release o passo acima deve ser feito também para a build configuration Release

4 - Fazer o mesmo para todos os projetos. Depois disto todos os builds devem estar na pasta ~ATSP/_builds/<pasta do projeto> e as libs na pasta ~ATSP/_libs

5 - Parâmetros na linha de comando são passados, para cada projeto, na aba projects à esquerda (Projects->Run)
